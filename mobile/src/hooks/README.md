# Custom Hooks

This directory contains custom React hooks.

## Suggested Hooks

- **useVoice**: Voice recognition hook
- **useTTS**: Text-to-speech hook
- **useStorage**: AsyncStorage hook
- **useNetwork**: Network state hook
- **useKeychain**: Secure storage hook

## Best Practices

- Follow React hooks naming convention (use prefix)
- Keep hooks focused and reusable
- Add TypeScript types
- Document hook parameters and return values
- Write tests for hooks
